# Icon128.png — Required Plugin Icon

This file is a placeholder. The plugin expects a 128×128 pixel PNG at this location.

To create one:
1. Open any image editor (Preview, GIMP, Photoshop).
2. Create a 128×128 canvas with a dark background.
3. Add text or an icon representing "emotion" (e.g., a waveform or face).
4. Export as `Icon128.png` and place it in this folder.

The plugin will compile and run correctly without this file, but the Epic Games Launcher
and the Plugin Browser will show a generic icon instead.
